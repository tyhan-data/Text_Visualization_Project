# 🚀 Text Data Visualization Project

## 📌 Project Overview

This project explores **text data analysis and visualization** using Python and common data science tools.

The project demonstrates how raw text can be transformed into meaningful numerical representations and visual insights using **Natural Language Processing (NLP)** techniques.

The dataset used is the **20 Newsgroups Dataset**, focusing on the **sci.space** category, along with a small demo text dataset.

This project highlights how data scientists analyze textual information and uncover patterns using visualization and dimensionality reduction techniques.

---

# 🎯 Project Objectives

The main goals of this project are:

* Perform **text preprocessing**
* Convert text into numerical form using **CountVectorizer**
* Analyze **word frequency patterns**
* Visualize text using **WordCloud**
* Reduce dimensionality using **PCA**
* Discover clusters using **t-SNE visualization**

# 📊 Visualizations Included

## 1️⃣ WordCloud Visualization

Displays the most frequently occurring words in the dataset.
Larger words represent higher frequency.

---

## 2️⃣ PCA Visualization

Principal Component Analysis (PCA) reduces high-dimensional text data into **two principal components** so patterns become easier to visualize.

---

## 3️⃣ t-SNE Visualization

t-SNE projects high-dimensional data into **2D space**, allowing us to visually detect clusters and similarities between text documents.
